# Gym Management System

A web-based Gym Management System developed using Flask and MySQL to efficiently manage gym operations, including members, trainers, memberships, attendance, and payments.

## Features

- Member Management
- Trainer Management
- Membership Plans
- Attendance Tracking
- Payment Management
- Secure Login System
- Responsive User Interface

## Tech Stack

### Backend
- Python
- Flask

### Database
- MySQL

### Frontend
- HTML
- CSS
- JavaScript

## Installation

1. Clone the repository

```bash
git clone https://github.com/azhaan355/gym-management-system.git
```

2. Navigate to the project directory

```bash
cd gym-management-system
```

3. Install dependencies

```bash
pip install -r requirements.txt
```

4. Configure the MySQL database.

5. Run the application

```bash
python app.py
```
